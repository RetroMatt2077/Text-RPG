# 🏰 RPG Toolkit Collection

A powerful collection of Python tools and games designed for tabletop RPGs, dungeon masters, and players who love world-building and adventure!

## 📋 Tools Included

| Tool                        | Purpose                          | Key Features |
|----------------------------|----------------------------------|--------------|
| `dice_roller_deluxe_v2.py` | Advanced Dice Roller            | Multiple dice types, advantage/disadvantage, roll history |
| `character_creator_v2.py`  | Character Builder               | Custom stats, classes, backgrounds, equipment |
| `npc_generator.py`         | NPC Generator                   | Random NPCs with names, personalities, backstories |
| `loot_generator.py`        | Loot & Treasure Generator       | Randomized items, weapons, magic gear, gold |
| `dungeon_generator.py`     | Dungeon Creator                 | Procedural dungeon maps and room descriptions |
| `calculator_deluxe.py`     | RPG Calculator                  | Damage, XP, gold calculations, and more |
| `retro_matt_rpg_deluxe.py` | Full Text-Based RPG             | Complete adventure with combat and progression |
| `text_rpg.py`              | Classic Text RPG                | Simple story-driven RPG experience |

## ✨ Highlights

- **All-in-one toolkit** for solo play, DM prep, or group campaigns
- **Procedural generation** for endless replayability
- **Easy to extend** — great for learning Python while building RPG tools
- Clean, commented code with fun flavor text
